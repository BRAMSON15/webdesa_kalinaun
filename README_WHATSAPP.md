# 📱 Notifikasi WhatsApp - Quick Guide

## 🚀 Cara Pakai (Super Simpel!)

### Untuk Admin - Notifikasi Bansos:
1. Buka **Bansos** → **Kelola Penerima**
2. Klik **Setujui** atau **Tolak**
3. Klik tombol **WhatsApp** (hijau/merah)
4. WhatsApp terbuka → Klik **Send** ✅

### Untuk Kades - Notifikasi Surat:
1. Buka **Validasi Pengajuan** → **Detail**
2. Pilih **Setujui** atau **Tolak**
3. Klik **Proses Pengajuan**
4. Klik tombol **Kirim Notifikasi WhatsApp**
5. WhatsApp terbuka → Klik **Send** ✅

## ✨ Fitur

- ✅ **GRATIS 100%** - Tidak perlu bayar API
- ✅ **Mudah** - Klik tombol → WhatsApp terbuka → Send
- ✅ **Aman** - Pakai WhatsApp resmi
- ✅ **Fleksibel** - Bisa edit pesan sebelum kirim

## 📋 Notifikasi yang Tersedia

### 1. Bansos
- ✅ Bansos Disetujui (dengan nominal)
- ❌ Bansos Ditolak (dengan alasan)

### 2. Pengajuan Surat
- ✅ Surat Selesai (dengan nomor surat)
- ❌ Surat Ditolak (dengan alasan)

## ⚠️ Syarat

- Masyarakat harus punya **nomor HP** di database
- Nomor HP harus **aktif WhatsApp**
- Admin/Kades harus punya **WhatsApp Web/Desktop**

## 🔧 Troubleshooting

### Tombol WhatsApp tidak muncul?
→ Cek nomor HP masyarakat di **Data Pengguna**

### WhatsApp tidak terbuka?
→ Install [WhatsApp Desktop](https://www.whatsapp.com/download)

### Nomor tidak valid?
→ Format: `08xxx` atau `62xxx`
